### Требования

- `git`
- `docker`
- `docker compose`
- `make`
---
### Установка

1. Склонировать репозиторий
```text
git clone <repo-url>
```

2. Собрать приложение
```bash
  make init
```

3. Запустить Octane
```bash
  make octane-up
```

После установки приложение будет доступно по адресу
```text
http://localhost:8000
```
---
### Хэлперы

Список доступных make таргетов
```bash
  make help
```
